# MV Engenharia & Construções Industriais

Website institucional premium desenvolvido para apresentar os serviços, projetos e diferenciais da MV Engenharia & Construções Industriais.

## Tecnologias utilizadas

* React / Next.js ou Vite
* TypeScript
* Tailwind CSS
* Framer Motion
* SEO otimizado
* Responsividade completa

## Funcionalidades

* Website institucional premium
* Seções de serviços
* Portfólio de obras
* Formulário de contato
* Integração com WhatsApp
* Animações modernas
* SEO técnico
* Layout responsivo
